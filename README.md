# Config.sketchybar

My sketchybar config.

![assets/Sketchybar Demo](./assets/Sketchybar_Demo.jpg)

## Fonts

```bash
brew install --cask sf-symbols
brew install --cask font-sf-mono
brew install --cask font-sf-pro
```

Application icons are built from [sketchybar-app-font](https://github.com/Jacky-Lzx/sketchybar-app-font).
